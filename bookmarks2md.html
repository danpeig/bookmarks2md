Bookmarks to Markdown

This tool converts exported bookmarks HTML from Browsers (Chrome, Firefox and others) to Markdown (.md) format.
It runs from a plain HTML file directly from the Browser used to create the bookmarks backup:
  • no installation
  • no data sent to any server - local processing
  • no extensions
  • no command line
  • no Python, Node, PHP or runtimes required
